# AI-Based Smart Waste Segregation System

This is a Semester 6 ESD project aligned with SDG 11, SDG 12, and SDG 13.

## Project Idea

The system classifies waste into four categories:

1. Organic / compostable waste
2. Recyclable waste
3. Hazardous / e-waste
4. Residual / mixed waste

The prototype accepts an image upload, observed item features, and an optional item name. It then applies AI-style scoring logic and gives the correct bin recommendation.

## How to Run

Open `index.html` in any modern browser.

No installation is required.

## Files

- `index.html` - main project demo
- `styles.css` - user interface design
- `script.js` - classification and recommendation logic
- `project-presentation.pptx` - presentation slides
